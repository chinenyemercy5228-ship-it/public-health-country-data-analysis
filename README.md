# 🌍 Public Health Country Data Analysis Using Pandas

## Overview

This project demonstrates how Python and Pandas can be used to clean, transform, and analyze real-world datasets.

The project was completed as part of my AI & Machine Learning learning journey through the TechRise program.

The goal was to take a raw country dataset, prepare the data, and extract meaningful insights through data analysis.

---

## Tools Used

- Python
- Pandas
- NumPy
- Jupyter Notebook

---

## Data Cleaning Process

The dataset was processed by:

- Importing CSV data into a dataframe
- Checking dataframe structure
- Inspecting columns and data types
- Renaming columns for better readability
- Converting data types
- Handling missing values
- Removing duplicate records

---

## Data Analysis Performed

The analysis included:

- GDP analysis
- GDP per capita classification
- Population analysis
- Country comparisons
- Continent-based aggregation
- Health expenditure analysis
- Education expenditure analysis
